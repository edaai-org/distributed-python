# Python 数据科学加速

开源的、面向下一代数据科学和人工智能应用的 Python 并行加速编程书籍。

## 内容介绍

Python 已经成为数据科学和人工智能引领性的编程语言，数据科学家常常使用 Python 完成一系列任务。本书主要针对 Python 无法高效并行所设计，重点以数据科学领域为应用场景。
